# Система интеллектуального распределения радиочастот на базе нечетких графов

Данный проект представляет собой интерактивное веб-приложение для автоматизации назначения частот приемопередатчикам. В основе лежит математический аппарат **теории нечетких графов**, позволяющий моделировать интерференцию сигналов не как бинарное событие, а как непрерывную величину (степень нечеткости).

## 1. Архитектура решения
Приложение построено на модульном принципе с использованием стека **Python/Streamlit**:
* **Ядро (`fuzzilib.py`):** Собственная библиотека для работы с нечеткими объектами. Реализует классы `FuzzyGraph`, `Node`, `Edge` и функции принадлежности.
* **Алгоритмический блок (`graph_algorithms.py`, `genetic_graph_algorithm.py`):** Набор решателей для задачи раскраски графа.
* **Интерфейс (`app.py`):** Реактивное веб-приложение для управления параметрами сети и визуализации.

## 2. Математическое моделирование интерференции
В приложении реализовано два метода расчета нечеткой связности между передатчиками $V_i$ и $V_j$:

### Метод А: Линейное затухание по расстоянию
Степень интерференции $\mu$ вычисляется относительно самого удаленного узла в сети:
$$\mu(e_{ij}) = 1 - \frac{D_{ij}}{D_{max}}$$

### Метод Б: Перекрытие зон покрытия
Моделирует физическое пересечение радиоволн:
$$\mu(e_{ij}) = 1 - \frac{D_{ij}}{r_i + r_j}$$

## 3. Методология решения ($\alpha$-срез)
Для перехода от нечеткой модели к техническому решению используется концепция **$\alpha$-среза**:
1. Пользователь задает порог $\alpha \in [0, 1]$.
2. Система формирует «четкий» подграф, отсекая связи, где $\mu < \alpha$.
3. Для полученного графа решается задача раскраски (назначение каналов).

## 4. Алгоритмы назначения частот (Раскраска графа)
Реализовано три метода, поддерживающих **индивидуальные ограничения** (запрещенные цвета для узлов):

### 4.1. Точный алгоритм (Backtracking)
* **Принцип:** Рекурсивный поиск с возвратом.
* **Оптимизация:** Вершины сортируются по убыванию степени.
* **Особенности:** Гарантирует минимальное число частот, но имеет высокую вычислительную сложность. Рекомендуется для $N < 30$.

### 4.2. Эвристический алгоритм (Welsh-Powell)
* **Принцип:** «Жадное» назначение цветов от наиболее связанных узлов к наименее связанным.
* **Особенности:** Работает мгновенно на больших графах. Адаптирован для обхода «дыр» в спектре, вызванных запрещенными каналами.

### 4.3. Генетический алгоритм
* **Принцип:** Стохастический поиск на основе популяции хромосом.
* **Fitness-функция:** Минимизирует конфликты, нарушения ограничений и ширину используемого спектра:
  $$F = 1000 \cdot N_{conflicts} + 1000 \cdot N_{forbidden} + 1 \cdot \max(Colors)$$

## 5. Анализ нечетких и темпоральных характеристик
* **Степень разделимости:** Оценка возможности бесконфликтной работы при дефиците частот.
* **Темпоральные графы:** Поддержка анализа динамических сетей (изменяющихся во времени) через поиск минимальной степени разделимости по всем временным срезам.

## 6. Визуализация
Используется **Pyvis** для интерактивного отображения:
* Толщина и прозрачность ребер зависят от уровня интерференции.
* Узлы окрашиваются в соответствии с назначенными частотами.
* Поддержка статического отображения на координатной сетке с радиусами покрытия.
